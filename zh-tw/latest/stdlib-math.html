數學模組¶

math 模組是一個提供數學函數的標準程式庫模組。使用以下方式匯入：

import math

然後以 math.isqrt(x)、math.sqrt(d) 等形式呼叫函數。

備註

import math 始終指向標準程式庫模組。若要匯入名為 math.vy 的本機檔案，請使用相對匯入：from . import math。

函數¶

sqrt(d: decimal) → decimal¶

使用巴比倫平方根演算法傳回所提供 decimal 數字的平方根。捨入模式為向下捨入到最接近的 epsilon。例如，math.sqrt(0.9999999998) == 0.9999999998。

import math

@external
@view
def foo(d: decimal) -> decimal:
    return math.sqrt(d)

>>> ExampleContract.foo(9.0)
3.0
isqrt(x: uint256) → uint256¶

使用巴比倫平方根演算法傳回所提供整數的（整數）平方根。捨入模式為向下捨入到最接近的整數。例如，math.isqrt(101) == 10。

import math

@external
@view
def foo(x: uint256) -> uint256:
    return math.isqrt(x)

>>> ExampleContract.foo(101)
10
